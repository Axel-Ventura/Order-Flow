# OrderFlow

## Descripción

OrderFlow es una aplicación web diseñada para ayudar a pequeños negocios a gestionar y dar seguimiento a sus pedidos de manera organizada.

El sistema permite registrar clientes, crear pedidos y controlar su estado dentro del proceso de venta, evitando la pérdida de información y mejorando la administración de las órdenes.

## Objetivo

Facilitar el control de pedidos mediante una plataforma sencilla y accesible para pequeñas empresas.

## Funcionalidades

- Inicio de sesión de usuarios.
- Gestión de clientes.
- Registro de pedidos.
- Consulta de pedidos.
- Actualización de estados.
- Control de acceso por roles.

## Estados de los pedidos

- Pendiente
- En Proceso
- Completado
- Cancelado

## Tecnologías

### Frontend
- React

### Backend
- Node.js
- Express

### Base de Datos
- MySQL

## Arquitectura

El sistema utiliza una arquitectura de tres capas:

1. Frontend (React)
2. Backend (Node.js + Express)
3. Base de Datos (MySQL)
